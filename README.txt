Smart Workout Builder

User would be able to create a new workout where he could choose his goal (or multiples):
- lose weight, gain muscle, gain endurance

He can also choose the number of days he would like to workout: 1-7

And the amount of time he would like his workouts to be: 10-90mins 

The application will then use Bayes Nets to build a workout plan for the user. The user 
will be able to save it and access it through the home screen when he would like to.
